# Expense Tracker

A full-stack Expense Management System built with **React + Vite** (Frontend) and **Node.js + Express + MongoDB** (Backend).

## 🚀 Tech Stack

- **Frontend:** React 18, Vite, React Router DOM, Chart.js, TailwindCSS
- **Backend:** Node.js, Express, MongoDB (Mongoose)
- **State Management:** React Context API
- **Database:** MongoDB Atlas + localStorage (frontend JSON layer)

## 📁 Project Structure

```
expence-tracker/
├── backend/          ← Node.js + Express API (PORT: 5000)
├── expense-tracker/  ← React + Vite Frontend (PORT: 5173)
├── README.md
└── package.json
```

## ⚡ Getting Started

### Install Dependencies
```bash
npm run install-backend
npm run install-frontend
```

### Run Backend
```bash
npm start
```

### Run Frontend
```bash
cd expense-tracker
npm run dev
```

Frontend runs at: **http://localhost:5173**  
Backend runs at: **http://localhost:5000**
